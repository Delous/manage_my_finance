# Finance CLI (in-memory)

Небольшое консольное приложение для учёта личных финансов.  
Хранит данные **только в памяти** (никаких БД и файлового хранилища для данных), поддерживает **несколько пользователей**, позволяет добавлять **доходы/расходы**, заводить **категории расходов** и задавать **лимиты (бюджет)** на категории. Есть вывод отчётов в консоль или в файл.

---

## Возможности

### Пользователи
- `register` — регистрация по логину и паролю
- `login` — вход по логину и паролю
- `logout` — выход из текущей сессии
- В приложении может существовать несколько пользователей, у каждого — свой кошелёк и свои категории/лимиты

### Финансы
- `income` — добавить доход (сумма + комментарий)
- `expense` — добавить расход (сумма + категория + комментарий)

### Категории и бюджет
- `cat-add` — создать категорию расходов
- `cat-list` — список категорий с лимитом/потрачено/остаток
- `budget` — установить лимит для категории

### Отчёты
- `summary` — общая сводка: доходы, расходы, баланс
- `by-cat` — отчёт по категориям: потрачено, лимит, остаток

### Куда печатать результаты
- `out` — переключение вывода:
    - `console` — печатать в терминал
    - `file` — писать в указанный файл (например `report.txt`)

---

## Ограничения и допущения

- Все данные живут **только в оперативной памяти** и пропадают при закрытии программы.
- Пароли не хранятся “как есть” — используется SHA-256 хэш (для учебного проекта, без соли и усложнений).
- Валюта не фиксируется: приложение работает с числами (BigDecimal), формат — как вводите.

## Структура проекта (кратко)

- `console/` — CLI, ввод команд, сессия, простая “табличка” для вывода
- `core/` — логика: авторизация, операции, отчёты
- `model/` — сущности (пользователь, кошелёк, операции, категории)

## Команды (шпаргалка)

> Команды вводятся без аргументов: приложение само спросит данные.

- `help` — список команд
- `register` — регистрация
- `login` — вход
- `logout` — выход
- `out` — выбрать вывод `console` или `file`
- `income` — добавить доход
- `expense` — добавить расход
- `cat-add` — добавить категорию
- `cat-list` — список категорий с текущими цифрами
- `budget` — установить лимит на категорию
- `summary` — сводка доходов/расходов/баланса
- `by-cat` — отчёт по категориям
- `exit` / `quit` — выход из программы

## Быстрый пример сценария

1) Регистрируемся:
- `register`
- логин: `delous`
- пароль: `1234`

2) Добавляем категорию и лимит:
- `cat-add` → `еда`
- `budget` → категория `еда`, лимит `5000`

3) Добавляем операции:
- `income` → сумма `20000`, комментарий (можно пусто)
- `expense` → сумма `350`, категория `еда`, комментарий (можно пусто)

4) Смотрим отчёты:
- `summary`
- `by-cat`

5) Пишем отчёт в файл:
- `out` → `file` → `report.txt`
- `by-cat` (результат уйдёт в `report.txt`)
